# Submission Checklist

- [ ] Указать автора
- [ ] Указать аффилиацию
- [ ] Указать email
- [ ] Выбрать журнал/конференцию
- [ ] Проверить требования журнала по объёму и структуре
- [ ] Добавить реальные источники по теме
- [ ] Оформить литературу в стиле целевого журнала
- [ ] Проверить таблицы и подписи в DOCX
- [ ] Вставить финальные рисунки или удалить placeholders
- [ ] Формализовать ручную проверку 35 пар (если возможно)
- [ ] Заменить placeholders в разделе доступности данных и кода
- [ ] Проверить раздел конфликта интересов
- [ ] Уточнить раздел финансирования
- [ ] Провести финальную языковую вычитку
- [ ] Проверить соответствие всех claims таблице статусов
- [ ] Проверить, что каждое численное утверждение имеет таблицу-источник
- [ ] Подготовить графики в требуемом журналом разрешении
